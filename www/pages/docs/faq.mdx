<FAQBox title="Is this RAG app free?">
- The code of this RAG app is open source, it's free to use;
- The database of this RAG app is free for 25GiB storage as we use [TiDB Serverless](https://tidb.cloud);
- **But** the token of OpenAI paid, you should have an account of platform.openai.com for an API token.
</FAQBox>

<FAQBox title="Can I use a different LLM?">
  Currently we use OpenAI by default, and can't change to another LLM.
  But we play to add more LLM support more LLM/LLM Platforms, such as: 

- [Claude3](https://claude.ai/)
- [Gemini](https://gemini.google.com/)
- [Ollama](https://ollama.ai/)
- ...

</FAQBox>
